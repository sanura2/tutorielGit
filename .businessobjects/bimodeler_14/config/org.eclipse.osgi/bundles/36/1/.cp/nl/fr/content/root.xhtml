Univers
L'Assistant de création d'univers vous assiste tout au long des étapes de création d'un univers relationnel ou OLAP.
Projet
Un projet est un espace de travail local dans lequel vous stockez les ressources qui servent à concevoir un ou plusieurs univers.
Connexion relationnelle
Une connexion relationnelle est un ensemble de paramètres qui définissent comment un univers peut accéder à une source de données relationnelle.
Connexion OLAP
Une connexion OLAP est un ensemble de paramètres qui définissent comment un univers peut accéder à un cube OLAP.
Fondation de données
Une fondation de données est un schéma des tables et jointures associées à une ou plusieurs bases de données relationnelles dans lesquelles des univers sont créés.
Couche de gestion
Une couche de gestion est un ensemble d'objets mappés sur la base de données relationnelle sous-jacente ou sur le cube OLAP.Vous nommez et organisez les objets de la couche de gestion de façon appropriée pour interroger l'univers.
