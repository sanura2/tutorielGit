Universe
The New Universe Wizard takes you through all the steps to create a relational or OLAP universe.
Project
A project is a local workspace where you store the resources used to build one or more universes.
Relational Connection
A relational connection is a set of parameters that define how a universe can access a relational data source.
OLAP Connection
An OLAP connection is a set of parameters that define how a universe can access an OLAP cube.
Data Foundation
A data foundation is a schema of the tables and joins from one or more relational databases on which you build one or more universes.
Business Layer
A business Layer is the set of objects that map to the underlying relational database or OLAP cube. You name and organize the business layer objects in a way that is meaningful for querying the universe.
